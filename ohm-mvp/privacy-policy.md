# Política de Privacidade — OHM MULTI SERVICES LDA

Última atualização: 2025-11-14

1. Coleta de dados
- Coletamos nome, telefone, e-mail e mensagem quando você envia um pedido de reserva ou contacto.

2. Uso dos dados
- Os dados são usados apenas para processar reservas, responder a solicitações e enviar confirmações.

3. Compartilhamento
- Não vendemos dados. Poderemos compartilhar informações com prestadores de serviços (ex.: serviços de email) exclusivamente para processar reservas.

4. Retenção
- Dados de reservas são mantidos durante o tempo necessário para cumprir a reserva e obrigações legais.

5. Direitos
- Os usuários podem solicitar acesso, correção ou remoção dos seus dados entrando em contacto via ohm.service.lda@gmail.com.

6. Contato
- ohm.service.lda@gmail.com
